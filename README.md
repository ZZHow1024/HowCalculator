## 它是什么？

**HowCalculator** 是一款在线计算器，它更关注于计算机组成原理相关的计算过程，现已支持 **补码一位乘法** 和 **数制转换**。

---

## 技术路线

- Web 框架：**React.js**
- 编程语言：**JavaScript**
